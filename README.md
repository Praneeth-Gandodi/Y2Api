# YTui
## Youtube Video/Audio Downloader

A simple Python tool to download videos and audio from YouTube.  
Built with **yt-dlp** and **Rich** libraries for progress display and better terminal UI.

---

## Note
⚠️ Do not use this tool for illegal or copyrighted content.  
⚠️ This project is still under development.  
⚠️ Videos are downloaded in their original quality and are **not compressed**. Some features may still have **bugs or issues**.
